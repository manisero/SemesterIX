Konwencja nazewnicza katalogów zawierających szczegółowe wyniki działania algorytmu dla kolejnych progów wsparcia:

sortX_storeY,
gdzie:
X - strategia sortowania (0 = Rosnące Wsparcie, 1 = Malejące wsparcie, 2 = Porządek leksykograficzny),
Y - strategia przechowywania identyfikatorów transakcji (0 = TIDSets, 1 = DiffSets).
